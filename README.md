# Tomodachi Life Decompilation

This is a WIP decompilation for Tomodachi Life for the 3DS.

## Setup

To setup the decompilation, copy the code.bin and exheader files from Tomodachi Life Revision 1. The SHA256 sum of the code.bin should be `e2ccc9e29349440bc128e712a87607d1fc941851a200016698e5592823720f32`.
In addition to the code.bin, a copy of armcc is required. See `tools/armcc/.gitkeep` for instructions.
