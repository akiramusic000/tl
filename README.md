# Tomodachi Life Decompilation

This is a WIP decompilation for Tomodachi Life for the 3DS.

## Setup

To setup the decompilation, copy the code.bin and exheader files from Tomodachi Life (USA Rev 1). The SHA256 sum of the code.bin should be `17ea0eee1d5c8ed4308ca58c021d4ddc116cf47abf7c369127c3115c5803b2f5`.
In addition to the code.bin, a copy of armcc is required. See `tools/armcc/.gitkeep` for instructions.
